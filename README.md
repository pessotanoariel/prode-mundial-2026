# Prode Mundial 2026

Proyecto personal para generar pronósticos del Mundial 2026 usando datos abiertos, criterios estadísticos simples y actualización progresiva con resultados reales.

## Objetivo

Construir un asistente de predicción para un prode del Mundial 2026, con foco en:

- carga de fixture;
- datos por selección;
- cálculo de fuerza relativa;
- predicción de resultado probable;
- actualización con resultados reales;
- evolución futura hacia modelos más avanzados.

## Stack inicial

- Python
- pandas
- numpy
